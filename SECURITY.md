## Reporting a Vulnerability

If you discover a security vulnerability in this project, please report it responsibly:

1. **Do not** open a public GitHub issue for security vulnerabilities.
2. Instead, please use [GitHub's private vulnerability reporting](https://github.com/dsanchezcr/ColonesExchangeRate/security/advisories/new) to submit your report.
3. Include as much detail as possible: steps to reproduce, affected versions, and potential impact.

### What to expect
- You will receive an acknowledgment within **48 hours**.
- We will work with you to understand and validate the issue.
- A fix will be developed and released as soon as possible, typically within **7 days** for critical issues.
- You will be credited in the release notes (unless you prefer otherwise).
